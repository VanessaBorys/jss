1) Kiedy powstał JavaScript (podaj datę).

JavaScript został opracowany w 1995 roku.

2) Dlaczego jest to język skryptowy?

JavaScript to język skryptowy, ponieważ przeglądarka może interpretować go bezpośrednio, bez konieczności wcześniejszej kompilacji.

3) Gdzie jest wykonywany JavaScript – po stronie klienta czy serwera?

JavaScript działa głównie po stronie klienta, czyli w przeglądarce, ale dzięki Node.js może być także uruchamiany na serwerze.

4) Dlaczego JavaScript jest bezpiecznym narzędziem?

JavaScript jest uważany za bezpieczny, ponieważ ma ograniczone uprawnienia dostępu do zasobów komputera, na którym działa.

5) Czy JavaScript jest językiem obiektowym? Jeśli tak, to dlaczego?

Tak, JavaScript jest językiem obiektowym, ponieważ umożliwia tworzenie obiektów i operowanie na nich, a także posługuje się metodami i zdarzeniami, co pozwala na pracę z modelem DOM.

6) Jakie są zalety JavaScriptu?
Zaleta 1: Jest prosty do nauki i szybki do przyswojenia.
Zaleta 2: Jest bardzo popularny – istnieje wiele przykładów i zasobów online.
Zaleta 3: Redukuje obciążenie serwera, wykonując część operacji po stronie użytkownika.
7) Co to jest framework?

Framework to zbiór narzędzi, który wspomaga proces tworzenia aplikacji, dostarczając struktury kodu oraz zestawu gotowych funkcji i bibliotek.

8) Czy można używać frameworków do aplikacji mobilnych?

Frameworki takie jak React Native umożliwiają tworzenie aplikacji mobilnych, które działają na różnych platformach.

9) Czym są aplikacje responsywne?

Aplikacje responsywne automatycznie dostosowują się do wielkości ekranu, co zapewnia poprawne wyświetlanie i działanie na różnych urządzeniach. JavaScript może pomagać w dynamicznym dopasowywaniu treści do ekranu.

10) Wyjaśnienie kodu: <input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')">

Ten kod tworzy przycisk, który po kliknięciu uruchamia funkcję WinOpen, otwierając nową stronę w nowym oknie przeglądarki.

11) Wyjaśnienie kodu: <window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");>

Ten kod otwiera nowe okno o nazwie "okienko" z plikiem "obraz.html". Ustawienia określają między innymi wyłączenie paska narzędzi, katalogów i menu oraz wymiary i pozycję okna na ekranie.

12) Co robi instrukcja window.close()?

Instrukcja window.close() zamyka obecnie otwarte okno przeglądarki.
